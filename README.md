# How to operate
- Run the requirments.txt file or just download the single lib

- Run the following command

```python main.py```

    1. Once opened create a urls.txt file which will be your housing for uTube musicvideosurls.

    2. Next select if its single urls (different songs,artist etc) or album. This is for file structure

    3. Next enter the genere. Again file structure reasoning

    4. Next Audio format (mp3,flac,m4a)

    5. Next select output folder

    6. Start download 🚀

```NOTE: After downloads are completed do not remove, the program will reallocate files into respected fodlers```
